General Information

  • Champ 101 starts and minimises automatically by default

  • After loading into a game you can access Champ 101 from the dock on top your screen

  • will not exit the app

  • Clicking on the game will hide the match!
    This behaviour is neccessary for Riot Games to allow the app and prevent unfair advantages through using the app.

App Walkthrough

  • Set the summoner name within the settings    to a player who is in a game (or try three dashes)

  • Open the match window by clicking  

  • Click a portrait in the match window to open a champions panel. You can open up to 6 champion panels for quick access

  • Within the champion panels, click on an ability to play its video (if available)

Known Issues

  • Incosistencies in public Data-sets:
    Values that we are unsure about are displayed with linethrough. They might still be correct, but don't bet on it. Where you see placeholders like {{ f3 }} no value could be mapped at all. Please do not report those cases, since we collect them already.

  • No data will be available for games with spectators disabled or for bots

  • Hotkeys will not work outside of the game and can not be the same for overwolf and game command

  • Some videos are not available in a supported format or the supported video might be out of date (for champion reworks)

  • Feel free to give feedback at the overwolf-store or send a message to champ101@krispin.it.

    If you're using Github, consider opening an issue for bugs and suggestions instead.
