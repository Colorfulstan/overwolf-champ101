Known Issues

  • No data will be available for games with spectators disabled or for bots

  • Champion data is gathered through riot's API.
    Since there are some inconsistencies within the data-sets, some values might not be correct or display their placeholder instead of a value. We did our best to work around those cases, but if you encounters errors of that type, feel free to send them as email so we can target those issues in future versions
