• Garena Servers now work

    Garena Players now can use Champ 101 too.

    Since Garena does not offer it's own data-API currently the data from Riots NA Servers is used within the app. That means the Leaue versions might not always match as they don't update at exactly the same time.

  • Fetching Summoner and Server automatically

    Summonername and Server don't have to be set manually anymore. If no RIOT-Server could be identified it is assumed that the game plays on a garena Server (Champion-data from NA will be used then).

    This Mechanic enables the App to fetch any kind of game, even with bots or with disabled spectators!

  • Changelog integrated
  • Replays and Spectates don't require Champ 101

    When spectating a game or your replays, the app will no longer start automatically.

    It is still possible to open the Match overview manually if you want to.

  • Version shown on Main Window
  • Correct Github Issue Link

    Now you actually can find the github-issues through the link.

  • The App now closes completely after a game ends

    There is no lingering Main-Window in the background anymore. The option to keep the Match open still works as before.
